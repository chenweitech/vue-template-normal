# vue-template-normal
## 基于Vue和Webpack4的企业中后台模版

## 目录结构

```
src
|- assets/        存放静态资源
|- components/    存放业务级别的UI组件
|- configs/       存放管理系统的配置文件
  |- SideBarConfig.json  侧边栏配置
|- layouts/       存放布局级别组件
|- pages/         存放页面级别组件
|- router/        存放页面的路由信息
|- styles/        存放全局的CSS样式
  |- variables.scss   存放全局的CSS变量
  |- mixin.scss       存放全局的混入样式
|- utils/         存放通用的功能性函数
|- App.vue        根组件
|- main.js        入口函数
```
